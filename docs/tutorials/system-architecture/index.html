Arquitetura do Sistema

A arquitetura computacional do PETROFÓLIO é composta por três principais componentes de software (veja a figura abaixo):

  • Interface Gráfica do Usuário (GUI) – Representada pelo suplemento do Excel.
  • Arquivos XML de entrada e saída de dados – Utilizados para armazenar e transferir informações.

  • Módulo de Cálculo do PETROFÓLIO – Responsável pela execução das análises econômicas.

Essa arquitetura flexível permite, por exemplo, a integração direta do módulo de cálculo com outros sistemas, tais como:

  • Bancos de dados
  • Simuladores de reservatórios
  • Sistemas de gestão de projetos, entre outros.
