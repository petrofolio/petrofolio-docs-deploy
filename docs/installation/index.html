Requisitos do Sistema

Para garantir o funcionamento adequado do PETROFOLIO, certifique-se de atender aos seguintes requisitos:

  • Sistema Operacional: Windows 10 ou superior.
  • Microsoft Excel: Versão desktop 2016 ou superior. Incompatível com o Office 365 Web.
  • Arquitetura: O Windows e o Excel devem ser de 64 bits (não compatível com 32 bits).

Download do Software

  • Acesse o site oficial:
    👉 download.PETROFOLIO.net.
  • Clique no botão “Baixar o instalador do PETROFOLIO” para iniciar o download do arquivo .zip.

Instalação por Usuário

Se deseja que o PETROFOLIO esteja disponível apenas para o usuário atual do computador, siga os passos abaixo.
⚠️ Importante: Este método não requer permissões de administrador.

Passo 1: Descompactar o Arquivo ZIP

  • Extraia o conteúdo do arquivo .zip para uma pasta local.
  • Certifique-se de que os seguintes arquivos estejam na pasta extraída:
    • setup.exe – Instalador principal do software.
    • PETROFOLIOAddInSetup.msi – Instalador do complemento do Excel.
    • Exemplo_Portfolio.xls – Arquivo de exemplo para uso no PETROFOLIO.

Passo 2: Executar o Instalador

  • Execute o arquivo setup.exe e siga as instruções na tela.
  • Recomendado: Utilize as configurações padrão para evitar problemas.





Passo 3: Concluir a Instalação

  • Após a finalização, feche a janela do instalador.
  • O PETROFOLIO estará pronto para uso.

Instalação Por Máquina

Se deseja que o PETROFOLIO esteja disponível para todos os usuários do computador, siga os passos abaixo.
⚠️ Importante: Este método requer permissões de administrador.

Passo 1: Descompactar o Arquivo ZIP

  • Extraia o conteúdo do arquivo .zip para uma pasta local.
  • Certifique-se de que os seguintes arquivos estejam presentes:
    • setup.exe – Instalador principal do software.
    • PETROFOLIOAddInSetup.msi – Instalador do complemento do Excel.
    • Exemplo_Portfolio.xls – Arquivo de exemplo.

Passo 2: Acessar o Prompt de Comando

  • Abra o Prompt de Comando (cmd.exe) como administrador.

Passo 3: Executar o Comando de Instalação

  • Navegue até a pasta onde o PETROFOLIO foi extraído, utilizando o comando cd:
     cd caminho-da-pasta-PETROFOLIO
  • Execute o comando abaixo para instalar o suplemento do Excel para todos os usuários:
    msiexec /i PETROFOLIOAddInSetup.msi ALLUSERS=1 TARGETDIR="C:\Program Files\PETROFOLIO\PETROFOLIOAddIn\"
