A quem se destina o PETROFOLIO?

O PETROFOLIO é uma ferramenta computacional desenvolvida para realizar análises econômicas de ativos de O&G no segmento E&P. Seu diferencial está na avaliação integrada de portfólios de ativos.

Seu uso pode atender a diversas necessidades, desde consultores e pesquisadores em instituições de ensino até companhias de petróleo, entidades certificadoras de reservas e auditorias.

Quais tipos de contratos de exploração e produção estão disponíveis no PETROFOLIO?

O PETROFOLIO contempla todos os contratos vigentes no Brasil, permitindo a inclusão de blocos sob diferentes regimes, como:

  • Concessão
  • Partilha de Produção
  • Cessão Onerosa
  • Áreas não contratadas da União

Regimes fiscais de outros países também podem ser adicionados sob demanda.

O PETROFOLIO é uma planilha do Excel?

Não. O PETROFOLIO é um software desenvolvido em C++, sendo o Excel utilizado apenas como interface gráfica para inserção de dados, execução de simulações e visualização de resultados. Todas as ferramentas são acessadas via um suplemento no Excel.

O PETROFOLIO pode ser customizado para atender às minhas necessidades?

Sim. A versão padrão do PETROFOLIO inclui as principais funcionalidades para avaliação econômica de ativos de E&P. No entanto, caso seja necessária a inclusão de critérios específicos, a ferramenta pode ser personalizada mediante solicitação.

Que tipo de suporte está disponível para usuários do PETROFOLIO?

Os usuários podem obter suporte por meio de diferentes canais:

  • Recursos de ajuda disponíveis na própria ferramenta.
  • Orientações no site: PETROFOLIO
  • Contato por e-mail: supporte@petrofolio.net

Como entrar em contato com os administradores do PETROFOLIO?

Para mais informações e suporte, acesse o site oficial ou entre em contato pelo e-mail:

  • Website: www.petrofolio.net
  • E-mail: supporte@petrofolio.net

O PETROFOLIO possui um site oficial?

Sim. O PETROFOLIO pode ser acessado pelo site:

  • www.petrofolio.net

No site, você encontrará informações sobre suas funcionalidades, características técnicas e modo de operação.

Existe treinamento para o uso do PETROFOLIO?

Sim. Além das instruções disponíveis na ferramenta e no site, é possível contratar treinamentos mais abrangentes, que podem ser:

  • Básicos – Para usuários iniciantes.
  • Personalizados – Com foco em funcionalidades específicas.

Quanto custa o PETROFOLIO?

O valor do PETROFOLIO varia conforme sua finalidade:

  • Gratuito para testes, instituições de ensino e pesquisa.
  • Pago para uso corporativo, com preços ajustados conforme o modelo de instalação (temporário, individual ou comunitário).

Como adquirir o PETROFOLIO?

As solicitações para obtenção do PETROFOLIO podem ser feitas diretamente no site oficial:

  • Acesse: PETROFOLIO

Como instalar o PETROFOLIO?

Após obter o instalador do PETROFOLIO, siga o procedimento padrão:

  1. Execute o arquivo de instalação.
  2. Siga as instruções na tela.
  3. Conclua a instalação e inicie o software.

Como ativar o PETROFOLIO para edição e simulação?

Após a instalação, será necessário ativar a ferramenta:

  1. Ao abrir o PETROFOLIO, um Código de Registro será gerado.
  2. Envie esse código para o e-mail suporte@petrofolio.net
  3. Você receberá um Código de Ativação, que deve ser inserido no software para liberar todas as funcionalidades.
